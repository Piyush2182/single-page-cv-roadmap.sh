# One Page HTML CV Project

This is a simple, single-page resume created using HTML and CSS. The goal of this project is to build a clean, professional-looking online CV without relying on any external frameworks or libraries.

## 🌐 Live Demo

[**View the live project here**](https://piyush2182.github.io/single-page-cv-roadmap.sh/)  
_(Note: Replace with your actual GitHub Pages URL after deployment)_

## 🚀 About This Project

This project is a personal resume page for Piyush Thakur, a Junior Frontend Developer. It includes sections for skills, education, work experience, and links to social profiles.

## ✨ Features

- Clean, minimalist design.
- Semantic HTML5 structure.
- Styled with an internal CSS stylesheet.
- Includes links for contact (`tel:`, `mailto:`) and social media profiles.

## 🛠️ Technologies Used

- **HTML5:** For the structure and content of the resume.
- **CSS3:** For styling the layout, typography, and colors.

## 📂 How to View

1.  Make sure all files (`index.html`, `favicon.png`) are in the same directory.
2.  Open the `index.html` file directly in any modern web browser (like Chrome, Firefox, or Edge).
